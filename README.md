# 📌 QAC-from-GitLab

## 📖 Overview

QAC-from-GitLab là một dự án Full-stack được xây dựng nhằm thực hành quy
trình phát triển phần mềm hiện đại, tổ chức cấu trúc project rõ ràng và
áp dụng kiến trúc RESTful.

Dự án tập trung vào việc xây dựng Back-end theo chuẩn Spring Boot, đồng
thời tách biệt rõ ràng giữa Frontend và Backend.

------------------------------------------------------------------------

## 🚀 Getting Started

### 1️⃣ Clone Repository

``` bash
git clone https://github.com/LuongHaMinhVy/qac-from-gitlab.git
cd qac-from-gitlab
```

------------------------------------------------------------------------

## ⚙️ Run Back-end

``` bash
cd back-end
mvn clean install
mvn spring-boot:run
```

Application mặc định chạy tại:

    http://localhost:8080

------------------------------------------------------------------------

## 💻 Run Front-end

``` bash
cd front-end
npm install
npm run dev
```

Front-end mặc định chạy tại:

    http://localhost:5173

------------------------------------------------------------------------

## 🔐 Main Features (Back-end Focus)

-   Xây dựng RESTful API theo chuẩn Spring Boot
